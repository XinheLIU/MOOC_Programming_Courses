# CS50: Introduction to Computer Science - Harvard University

**Provider:** Harvard University  
**Course Type:** Complete course with 8 problem sets

## Course Overview

CS50 is Harvard's introduction to computer science and programming. This course teaches students how to think algorithmically and solve problems efficiently. It covers multiple programming languages and culminates in a web development project.

## Target Audience

- Complete beginners to programming
- Students wanting computer science fundamentals
- Anyone wanting to learn programming properly

## Learning Objectives

- Learn to think like a computer scientist
- Master fundamental programming concepts
- Understand data structures and algorithms
- Build web applications

## Course Content

### Problem Sets (8 total)

1. **C Basics** - Variables, loops, functions
2. **Arrays & Strings** - Memory management in C
3. **Algorithms** - Search, sort, recursion
4. **Memory** - Pointers, dynamic allocation
5. **Data Structures** - Linked lists, trees, hash tables
6. **Python** - Transition to Python programming
7. **Web Development** - HTML, CSS, JavaScript, Flask
8. **Final Project** - Capstone web application

## Prerequisites

- None! This is designed for absolute beginners
- Willingness to learn and practice

## Content Structure

- `pset1/` through `pset8/` - Problem set directories
- `lecture0.pdf` through `lecture11.pdf` - Lecture slides
- Code examples and starter files
- Data files for exercises

## Languages Covered

- **C** - Systems programming (Weeks 1-5)
- **Python** - High-level programming (Week 6+)
- **SQL** - Database queries
- **HTML/CSS/JavaScript** - Web development
- **Flask** - Web framework

## How to Use

1. Watch lectures (PDFs provided)
2. Complete problem sets sequentially
3. Start with `pset1/` and work through to `pset8/`
4. Build final project in `pset8/`

## Key Topics Covered

- Programming fundamentals
- Data structures (arrays, linked lists, trees, hash tables)
- Algorithms (search, sort, recursion)
- Memory management
- Web development
- Database design
